Leia até o final antes de usar os scripts!

Instruções de uso:

1) Conecte seu HD externo ao computador e extraia dentro dele os conteúdos do arquivo zipado "DualBoot BKP".

2) Dê dois cliques nos arquivos BKP_ADD ou BKP_SYNC para iniciar seu backup. Uma janela do Prompt de comando pode abrir e passar letras bem rápido mas não se preocupe, ela fechará sozinha assim que o processo de backup terminar.

3) Verifique que foi criada uma pasta com o backup dos arquivos do seu usuário. 
A primeira vez que o script roda é a mais demorada porque todos os arquivos são copiados. Nas vezes seguintes apenas novos arquivos ou arquivos modificados serão copiados.

----------------------------------------------------------------

Diferenças entre BKP_ADD e BKP_SYNC: 

BKP_ADD copia do seu computador para o seu HD externo novos arquivos criados ou modificados (deletando a versão anterior que estava armazenada no HD). Arquivos que você deletou no seu computador continuarão no HD externo.

BKP_SYNC, por outro lado, além de copiar para o HD externo os arquivos que foram criados ou modificados no seu computador, deleta no HD os arquivos que você deletou no computador.

----------------------------------------------------------------

OBS: Se seu computador tiver sido especificamente configurado para direcionar seus arquivos pessoais para uma partição diferente do disco do sistema operacional, os scripts não funcionarão, não copiando os arquivos - não devem, no entanto, causar dano ao seu sistema ou arquivos.

OBS.2: Ao tentar executar os arquivos BKP_ADD e BKP_SYNC o Windows pode alertar sobre o arquivo ser perigoso, devido ao fato de o script ter uma origem desconhecida ao sistema. Você pode ignorar esses alertas mas, se quiser, pode abrir o script usando o bloco de notas para ver exatamente os comandos que ele executa no sistema.
